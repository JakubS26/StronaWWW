Moje projekty

  W czasie moich studiów, jak i w czasie wolnym udało mi się zrealizować sporo ciekawych projektów i zadań. Wśród nich znajdowały się na przykład:

  • aplikacja do gry w chińskie warcaby
  • prosty program do rysowania
  • aplikacja bazodanowa obsługi schroniska dla psów
  • solver do łamigłówki Sudoku
  • implementacje różnych metod kompresji bezstratnej
    (np. kodowanie arytmetyczne, algorytm LZW)
  • aplikacja webowa do gry w szachy
  • program do rozwiązywania problemu komiwojażera za pomocą metaheurystyk
  • prosta powłoka systemu Linux
Na tej stronie opisałem najciekawsze ze zrealizownych przeze mnie projektów.

Solver sudoku

  Program rozwiązuje sudoku używając algorytmu przeszukiwania z nawrotami. Za jego pomocą można wczytać plik opisujący diagram do rozwiązania, wyświetlić łamigłówkę przed rozwiązaniem a następnie z wpisanymi brakującymi liczbami. Jeśli dany przykład posiada więcej niż jedno rozwiązanie, to program znajdzie je wszystkie. Solver został napisany w języku C.

Aplikacja do gry w chińskie warcaby

  Aplikacja została stworzona w języku Java. Działa ona na zasadzie architektury klient-serwer. Umożliwia uruchomienie serwera dla 2, 3, 4 lub 6 graczy a następnie podłączenie się do niego i rozegranie partii. Serwer weryfikuje poprawność ruchów graczy, informuje ich, kiedy mają wykonać swoje posunięcie lub kiedy rozgrywka się zakończy.

Program rozwiązujący TSP za pomocą metaheurystyk

  Zadaniem programu jest znajdowanie przybliżonych rozwiązań problemu komiwojażera za pomocą różnych heurystyk i algorytmów metaheurystycznych. Możliwe jest wczytanie instancji problemu z biblioteki TSPLIB lub wygenerowanie losowej instancji. Wśród zaimplementowaych algorytmów były między innymi algorytm najbliższego sąsiada, 2-opt, przeszukiwanie tabu (tabu search) oraz algortym genetyczny. W przypadku instancji problemu, w których wierzchołki odpowiadają punktom na płaszczyźnie, możliwe było również zwizualizowanie znalezionego rozwiązania (na rysunku znajduje się rozwiązanie znalezione przez program dla instancji a280.tsp z biblioteki TSPLIB). Program został stworzony w Javie.

Aplikacja webowa do gry w szachy

  Projekt ten był tworzony w pięcioosobowej grupie w ramach przedmiotu Programowanie zespołowe. Częścią projektu, przy której najwięcej pracowałem było stworzenie własnego silnika szachowego, który działa na podstawie algorytmu min-max z użyciem tzw. alfa-beta obcięć. Silnik powstał w języku Java, a następnie został przystosowany do działania na zasadzie API internetowego za pomocą frameworka Spring. Obok znajduje się kawałek kodu odpowiadający za wyznaczanie ruchów wieży.

private static void addRookMoves(Board board, int beginCol, int beginRow, PieceColor myColor, List<Move> possibleMoves) {
 PieceColor oppColor = opponentColor(myColor);

 int i = 1;
 while (onChessboard((beginCol + i), beginRow) && freeSquare(board, (beginCol + i), beginRow)) {
  possibleMoves.add(new Move(beginCol, beginRow, (beginCol + i), beginRow));
  i++;
 }
 if (onChessboard((beginCol + i), beginRow)) {
  if (board.squares[(beginCol + i)][beginRow].getColor() == oppColor) {
   possibleMoves.add(new Move(beginCol, beginRow, (beginCol + i), beginRow));
  }
 }

 i = 1;
 while (onChessboard((beginCol - i), beginRow) && freeSquare(board, (beginCol - i), beginRow)) {
  possibleMoves.add(new Move(beginCol, beginRow, (beginCol - i), beginRow));
  i++;
 }
 if (onChessboard((beginCol - i), beginRow)) {
  if (board.squares[(beginCol - i)][beginRow].getColor() == oppColor) {
   possibleMoves.add(new Move(beginCol, beginRow, (beginCol - i), beginRow));
  }
 }

 i = 1;
 while (onChessboard(beginCol, (beginRow + i)) && freeSquare(board, beginCol, (beginRow + i))) {
  possibleMoves.add(new Move(beginCol, beginRow, beginCol, (beginRow + i)));
  i++;
 }
 if (onChessboard(beginCol, (beginRow + i))) {
  if (board.squares[beginCol][(beginRow + i)].getColor() == oppColor) {
   possibleMoves.add(new Move(beginCol, beginRow, beginCol, (beginRow + i)));
  }
 }

 i = 1;
 while (onChessboard(beginCol, (beginRow - i)) && freeSquare(board, beginCol, (beginRow - i))) {
  possibleMoves.add(new Move(beginCol, beginRow, beginCol, (beginRow - i)));
  i++;
 }
  if (onChessboard(beginCol, (beginRow - i))) {
   if (board.squares[beginCol][(beginRow - i)].getColor() == oppColor) {
  possibleMoves.add(new Move(beginCol, beginRow, beginCol, (beginRow - i)));
  }
 }
}
